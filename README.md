# JavaScript Basics

Repositório com exercícios básicos de JavaScript focados em lógica de programação,
manipulação do DOM e validações simples.

## Conteúdo

- Exercícios de lógica
- Manipulação básica do DOM
- Eventos
- Validações simples

## Projetos

### Age Checker
Verificador de idade com entrada de dados e exibição de resultado no DOM.

### Counter
Contador simples utilizando eventos e manipulação do DOM.

### Simple Calculator
Calculadora básica com operações matemáticas simples.

## Tecnologias
- HTML5
- JavaScript (Vanilla)

## Objetivo
Praticar os fundamentos de JavaScript para desenvolvimento front-end.
